# JavaScript 装饰器 API 中文文档

本项目收录了 2ality 网站关于 JavaScript 装饰器 API 的相关文档，并进行了中文翻译。

## 项目结构

- `JavaScript2022-03decoratorsAPI.html`：极为中英双语文章，鼠标指在文本上点击可以切换，也可以右上角按钮切换语言

## 使用方法

直接用浏览器打开对应的 HTML 文件即可阅读。
